# this

In JavaScript, there is a very interesting keyword called `this`. We are going to learn it later in 
details. In this excercise, you just simply type `this` to the REPL and see what you get.

Note: When post your result, please double check and remove sensitive private information such as

```
GITHUB_TOKEN: '****************************************',
```
